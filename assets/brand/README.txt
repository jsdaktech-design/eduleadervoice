EduLeader Voice — Brand / Logo Pack
===================================

The mark: a Didone serif "E" (EduLeader) with a brass quotation mark (Voice),
set in a double hairline ring — an editorial / atelier seal.

Palette
-------
Pine   #1e3a32   (primary dark)
Ivory  #f4f1e8   (paper / light)
Brass  #9c7b3f   (accent; lighter tint #c4a062)

USAGE NOTES
-----------
• Prefer the SVGs for web and print — they stay crisp at any size and the lockup
  renders in the brand typeface (Fraunces). The lockup PNGs use a serif fallback
  for the wordmark, so reach for the SVG when the type must be exact.
• On dark or photographic backgrounds use the -ivory variants.
• Keep clear space around the mark equal to roughly half the seal's diameter.
• Don't recolour, stretch, rotate, or add effects to the mark.
